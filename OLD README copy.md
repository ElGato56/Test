# jspsych-foraging
For the development of a jsPsych pluging for visual foraging tasks

These packages need to be installed first:
Visual Studio Code: https://code.visualstudio.com/
Node.js: https://nodejs.org
Git: https://git-scm.com/

- Open Visual Studio Code, click on the button "Source Control" in the left sidebar (third element from top, looks like a fork), click on "Clone Repository"
- Copy repository URL from Github website and paste into text field
- Open Terminal in Visual Studio Code in menu bar -> New Terminal, paste this:
- npm install
- npx jspsych run -e EXPERIMENT NAME (replace with name of experiment you want to run)
- Open a browser and go to localhost:3000!
